## 🎁 특별 혜택

### "1명 구독 = 길드 전체 혜택!"

→ 전체 길드원이 카카오톡 알림을 받을 수 있어요!

```
길드원 10명 중 1명이 ₩2,500 구독

```

## 📱 핵심 기능: 카카오톡 알림

### 왜 카톡 알림이 중요한가요?

❌ 디스코드를 항상 켜놓기 어려워요  
❌ 디스코드 알림을 놓쳐요  
❌ 폰으로 보기 불편해요  

✅ 카톡은 항상 켜져있어요!  
✅ 카톡 알림은 절대 안 놓쳐요!  
✅ 폰으로 보기 편해요!  

### 카톡으로 받는 것

**1. 매일 아침 일정 요약 (카톡방)**
```
[로일봇]
━━━━━━━━━━━━━━
📅 오늘(월요일) 일정
━━━━━━━━━━━━━━

20:00 종막 하드1
 └ 거니, 자두, 하즈, 김실순

21:00 세르카 나메1
 └ 츄츄캉, 뒤버, 유라이니

오늘도 화이팅! 💪
━━━━━━━━━━━━━━
```

**2. 레이드 알림 (개인 DM)**
```
━━━━━━━━━━━━━━
🔔 거니님 알림
━━━━━━━━━━━━━━
30분 후 레이드!

종막 하드1 | 20:00
역할: 1파티 서폿
캐릭터: 빛쟁인거니

준비해주세요! 💪
━━━━━━━━━━━━━━
```

### 카톡 알림 설정 방법

#### Step 1: 프리미엄 구독
(아래 구독 방법 참고)

#### Step 2: 코드 발급
디스코드에서:
```
/카톡봇코드
```

**출력:**
```
✅ 카톡봇 연동 코드 발급!

━━━━━━━━━━━━━━
코드: ABC-123-XYZ
━━━━━━━━━━━━━━

📱 사용 방법:
1. 카카오톡에서 '로일봇' 검색
2. 친구 추가
3. 코드 입력
4. 길드 카톡방에 초대
```

#### Step 3: 카톡 연동

1. 카카오톡 열기
2. 검색창에 **"로일봇"** 입력
3. 친구 추가
4. 로일봇과 1:1 채팅
5. 코드 입력: `ABC-123-XYZ`

#### Step 4: 카톡방 초대

1. 길드 카톡방 열기
2. 우측 상단 ≡ 메뉴
3. 대화상대 초대
4. **"로일봇"** 선택

### ✅ 완료!
이제 전체 길드원이 카톡 알림을 받습니다! 🎉

## ❓ 자주 묻는 질문

### Q. 무료로도 충분한가요?
A. 네! 핵심 기능은 모두 무료입니다. 프리미엄은 선택사항입니다.

### Q. 카톡 알림 때문에 구독하려는데?
A. 좋은 선택입니다! 길드에 1명만 구독하면 전체가 혜택을 받으니 부담도 적어요.

### Q. 구독 취소하면 어떻게 되나요?
A. 무료 플랜으로 자동 전환됩니다. 기존 데이터는 모두 유지됩니다.

### Q. 환불이 가능한가요?
A. 7일 이내 전액 환불, 7일 이후 비례 환불이 가능합니다.

### Q. 결제 실패하면?
A. 3일 유예 기간이 있습니다. 기간 내 결제하시면 서비스가 유지됩니다.

### Q. 길드 단위 구독 시 혜택은?
A. 지정한 15명의 길드원이 모두 라이트 플랜을 사용할 수 있습니다.

### Q. API 키 5개가 꼭 필요한가요?
A. 대형 길드(40명 이상)가 아니면 3개로도 충분합니다. 5개는 더 빠른 속도를 원하시는 분들께 추천합니다.
