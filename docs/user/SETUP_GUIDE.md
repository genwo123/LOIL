# 🛠 로일 설치 가이드

5분이면 완료! 차근차근 따라해보세요 😊

## 📋 준비물

시작하기 전에 다음을 준비해주세요:

✅ 디스코드 서버 (관리자 권한 필요)  
✅ 구글 계정  
✅ 로스트아크 캐릭터  
✅ 로스트아크 API 키 2-3개 (발급 방법 아래 참고)

## 🚀 Step 1: 봇 초대하기 (1분)

### 1-1. 초대 링크 클릭
[로일 봇 초대하기](링크 예정) ← 클릭!

### 1-2. 서버 선택
![서버 선택 화면]

1. 봇을 추가할 디스코드 서버를 선택하세요
2. "계속하기" 버튼 클릭

### 1-3. 권한 승인
![권한 승인 화면]

필요한 권한을 확인하세요:
- 메시지 읽기/보내기
- 임베드 링크
- 멘션하기
- 채널 관리 (자동 채널 생성용)

"승인" 버튼 클릭

### ✅ 완료!
로일 봇이 서버에 입장했습니다! 🎉

봇이 자동으로 다음 채널들을 생성합니다:
- 📢 로일-알람
- 📅 레이드-일정
- 📊 통계
- ⚙️ 로일-설정

---

## ⚙️ Step 2: 초기 설정 (2분)

### 2-1. 설정 시작
디스코드 채팅창에 입력:
```
/설정
```

### 2-2. 안내 따라하기
봇이 친절하게 안내해드립니다!

**로일봇:**
```
안녕하세요! 로일 설정을 시작합니다 🎮

1️⃣ 길드 정보를 입력해주세요
   - 길드명: 빛쟁이들
   - 서버: 아만

2️⃣ 알림 채널을 선택해주세요
   - #로일-알람 (자동 생성됨)

3️⃣ 시간대를 설정해주세요
   - 기본: 서울(KST)

✅ 설정 완료!
```

---

## 🔑 Step 3: API 키 등록 (3분) ⭐ 중요!

### 3-1. API 키 발급받기

1. [로스트아크 개발자 센터](https://developer-lostark.game.onstove.com/) 접속
2. 로그인 (Stove 계정)
3. "API 키 발급" 클릭
4. 키 복사

💡 **팁:** 길드원들과 나눠서 발급받으세요!
- 길마: 1개
- 부길마: 1개
- 레이드장: 1개
→ 총 3개!

### 3-2. 디스코드에서 키 등록

**무료 길드 (최대 3개):**
```
/api추가 [API키1]
/api추가 [API키2]
/api추가 [API키3]
```

**로일봇:**
```
✅ API 키 1/3 등록 완료!
별명: 길마 키

✅ API 키 2/3 등록 완료!
별명: 부길마 키

✅ API 키 3/3 등록 완료!
별명: 예비 키

🚀 총 3개 키 등록!
이제 빠른 속도로 이용 가능합니다.
```

### 3-3. 키 상태 확인
```
/api상태
```

**로일봇:**
```
📊 API 키 상태

1. 길마 키: ✅ 정상 (60/60 남음)
2. 부길마 키: ✅ 정상 (60/60 남음)
3. 예비 키: ✅ 정상 (60/60 남음)

💡 모든 키가 정상입니다!
```

## 📊 Step 4: 구글 시트 연동 (2분)

### 4-1. 템플릿 복사

1. [구글 시트 템플릿 열기](링크 예정)
2. 상단 메뉴: **파일 → 사본 만들기**
3. 이름 변경: "빛쟁이들 레이드 일정"

![템플릿 복사 화면]

### 4-2. 시트 구조 확인
복사한 시트에는 다음 탭이 있습니다:
```
📑 시트 구성
├── 데이터베이스 (길드원, 캐릭터 정보)
├── 거니 (개인 일정)
├── 자두 (개인 일정)
├── 주간레이드 (전체 일정 요약)
└── AI추천 (AI가 자동 생성)
```

### 4-3. 길드원 정보 입력
"데이터베이스" 탭에 입력:

| 플레이어 | 캐릭터명 | 직업 | 레벨 | 역할 | 상태 |
|---------|---------|------|------|------|------|
| 거니 | 빛쟁인거니 | 홀나 | 1750 | 서폿 | 활성 |
| 자두 | 자두바드 | 바드 | 1745 | 서폿 | 활성 |

![데이터베이스 입력 예시]

### 4-4. 시트 공유하기

1. 우측 상단 **"공유"** 버튼 클릭
2. **"링크가 있는 모든 사용자"** 선택
3. 권한: **"편집자"** 또는 **"뷰어"** (추천: 뷰어)
4. **"링크 복사"** 클릭

![시트 공유 화면]

### 4-5. 디스코드 연동
디스코드에서 입력:
```
/시트연동 [복사한 링크 붙여넣기]
```

**예시:**
```
/시트연동 https://docs.google.com/spreadsheets/d/1ABC...
```

**로일봇:**
```
✅ 구글 시트 연동 완료!
10명의 길드원 정보를 불러왔습니다.
```

## ❓ 문제 해결 (Troubleshooting)

### 봇이 응답하지 않아요

**체크리스트:**
- [ ] 봇이 서버에 있나요?
- [ ] 봇이 채널을 볼 수 있나요? (권한)
- [ ] 명령어가 올바른가요? (슬래시 명령어)
- [ ] 서버 상태가 정상인가요?

**해결 방법:**
1. 봇을 재초대해보세요
2. `/동기화` 명령어 실행
3. 지원 서버에 문의

### 구글 시트가 연동 안 돼요

**체크리스트:**
- [ ] 시트가 공유 설정되어 있나요?
- [ ] 링크가 올바른가요?
- [ ] 시트 구조가 템플릿과 같나요?

**해결 방법:**
1. 시트 공유 설정 다시 확인
2. 템플릿에서 다시 복사
3. `/시트연동` 다시 시도

### 캐릭터 등록이 안 돼요

**체크리스트:**
- [ ] 캐릭터명이 정확한가요?
- [ ] 로스트아크 API가 정상인가요?
- [ ] 최근에 캐릭터를 만들었나요? (24시간 대기)
- [ ] API 키가 등록되어 있나요?

**해결 방법:**
1. 캐릭터명 다시 확인
2. API 키 상태 확인 (`/api상태`)
3. 수동으로 시트에 입력
4. 잠시 후 다시 시도

### 알림이 오지 않아요

**체크리스트:**
- [ ] 알림 설정이 켜져 있나요?
- [ ] DM 설정이 열려있나요?
- [ ] 일정이 등록되어 있나요?

**해결 방법:**
1. `/내알림` 으로 설정 확인
2. 디스코드 개인정보 설정 확인
3. `/알림설정` 다시 설정

### API 키 에러가 떠요

**체크리스트:**
- [ ] API 키가 올바른가요?
- [ ] API 키가 만료되지 않았나요?
- [ ] 모든 키가 Rate Limit에 걸렸나요?

**해결 방법:**
1. `/api상태`로 키 상태 확인
2. 새 API 키 추가
3. 잠시 후 재시도 (Rate Limit 리셋 대기)
