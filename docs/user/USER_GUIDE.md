# 📖 로일 사용 가이드

로스트아크 길드 레이드 일정을 AI가 자동으로 관리해드려요!

---

## 🎯 로일이란?

**로일(LoIl)**은 로스트아크 길드의 레이드 일정을 쉽고 편리하게 관리할 수 있도록 도와주는 디스코드 봇입니다.

### 이런 분들께 추천해요

✅ 매주 레이드 일정 짜기 힘드신 길드 마스터님  
✅ 누가 언제 참여 가능한지 파악하기 어려우신 분  
✅ 서폿 배치가 매번 고민되시는 분  
✅ 길드원들과 소통이 번거로우신 분  

---

## ✨ 주요 기능

### 1. 📊 구글 시트로 간편하게
익숙한 스프레드시트로 일정을 관리하세요!

- 엑셀처럼 쉽게 편집
- 길드원 모두가 함께 작성
- 실시간 자동 동기화

### 2. 🤖 AI가 자동으로 파티 편성
복잡한 일정 조율을 AI가 대신해드려요!

- 서폿 자동 배치
- 시간대별 최적화
- 공방 필요 인원 계산
- API 키 자동 관리 (무료: 3개, 프리미엄: 5개)

### 3. 🔔 놓치지 않는 알림
레이드 시간을 잊지 않도록!

- 디스코드 멘션/DM
- 원하는 시간에 알림 (30분 전, 15분 전 등)
- **프리미엄**: 카카오톡 알림까지!

### 4. 📈 상세한 통계
우리 길드 활동을 한눈에!

- 레이드 참여 기록
- 클리어 타임 분석
- 골드 수익 계산

## 🚀 시작하기

### 1단계: 봇 초대하기

1. [봇 초대 링크](링크 예정)를 클릭하세요
2. 권한을 승인해주세요
3. 디스코드 서버에 봇이 들어옵니다!

### 2단계: 초기 설정
디스코드에서 다음 명령어를 입력하세요:
```
/설정
```
봇이 안내하는 대로 따라하시면 됩니다!

### 3단계: 구글 시트 연동

1. [구글 시트 템플릿](링크 예정)을 복사하세요
2. 길드원 정보를 입력하세요
3. 시트 공유 링크를 복사하세요
4. 디스코드에서 연동하세요:
```
/시트연동 [복사한 링크]
```

### 4단계: 캐릭터 등록
```
/원정대등록 [내 캐릭터명]
```
로스트아크 API가 자동으로 내 모든 캐릭터를 불러옵니다!

### 5단계: API 키 등록 (중요!)
각 길드는 로스트아크 API 키를 등록해야 합니다.

**무료 길드:**
```
/api추가 [API키1]
/api추가 [API키2]
/api추가 [API키3]
```
최대 3개까지 등록 가능!

**프리미엄 길드:**
```
/api추가 [API키4]
/api추가 [API키5]
```
최대 5개까지 등록 가능!

💡 **왜 여러 개?**
- 한 키당 분당 100회 제한
- 여러 키로 요청 분산
- 빠른 응답 속도

```

### 통계
```

```
/알림설정 DM 30 15    # 30분, 15분 전 DM 알림

/api상태           # API 키 상태 확인
/api삭제 1         # 1번 키 삭제
```

## 📱 카카오톡 알림 (프리미엄)

### 설정 방법

1. **프리미엄 구독**
2. **디스코드에서 코드 발급:**
```
/카톡봇코드
```

3. **카카오톡에서 "로일봇" 검색** → 친구 추가
4. **코드 입력**
5. **길드 카톡방에 봇 초대**

### 카톡으로 받는 것
- 📅 매일 아침 9시: 오늘 일정 요약
- 🔔 레이드 30분 전: 개인 DM 알림

## 🔑 API 키 관리 팁

### API 키 발급 방법
1. [로스트아크 개발자 센터](https://developer-lostark.game.onstove.com/) 접속
2. 로그인 후 API 키 발급
3. 여러 계정으로 2-3개 키 발급 (친구, 길드원)

### 왜 여러 개가 좋나요?
```
1개 키: 100 요청/분 → 길드 30명 사용 시 느림
3개 키: 300 요청/분 → 3배 빠름! ✨
5개 키: 500 요청/분 → 5배 빠름! 🚀 (프리미엄)
```

### 자동 관리
- 봇이 알아서 키를 돌아가며 사용 (Round-Robin)
- 한 키가 제한되면 자동으로 다른 키 사용
- 키 상태 실시간 추적

## ❓ 자주 묻는 질문 (FAQ)

### Q. 무료인가요?
A. 네! 핵심 기능은 모두 무료입니다.

### Q. 프리미엄 꼭 구독해야 하나요?
A. 아니요! 무료로도 충분히 사용 가능합니다.

### Q. API 키는 꼭 필요한가요?
A. 네, 로스트아크 API를 사용하려면 필수입니다.

### Q. API 키 발급이 어려워요
A. 길드원들과 나눠서 발급받으세요! 각자 1개씩만 발급해도 됩니다.

### Q. 다른 게임도 지원하나요?
A. 현재는 로스트아크만 지원하고, 향후 WoW 등 추가 예정입니다.

### Q. 길드원이 많으면 어떻게 하나요?
A. 인원 제한 없이 사용 가능합니다! API 키만 충분하면 됩니다.

### Q. 데이터가 안전한가요?
A. 모든 데이터는 안전하게 암호화되어 저장됩니다.

### Q. 봇이 응답하지 않아요
A. 1) 봇 권한 확인 2) `/동기화` 명령어 실행 3) 그래도 안 되면 지원 서버로 문의해주세요

### Q. 시트를 잘못 수정했어요
A. 걱정 마세요! 구글 시트의 버전 기록에서 복구할 수 있습니다.

### Q. 길드 마스터만 설정할 수 있나요?
A. 초기 설정은 관리자 권한이 필요하지만, 이후에는 모두가 일정을 입력할 수 있습니다.
