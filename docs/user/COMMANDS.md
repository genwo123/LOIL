## 🔔 알림 설정

### /알림설정 [방식] [시간들]
알림 방식과 시간을 설정합니다.

**사용법:**
```
/알림설정 DM 30 15 5
/알림설정 태그 30
/알림설정 둘다 30 15
/알림설정 끄기
```

**파라미터:**
- `방식` (필수): DM, 디엠, 태그, 멘션, 둘다, 끄기
- `시간들` (선택): 분 단위 (예: 30 = 30분 전)

**방식:**
- `DM` 또는 `디엠`: 개인 DM
- `태그` 또는 `멘션`: 채널 @멘션
- `둘다`: 채널 + DM
- `끄기`: 알림 끄기

**예시 출력:**
```
✅ 알림 설정 완료!

방식: 개인 DM
시간: 30분, 15분, 5분 전
```

**권한:** 모든 사용자  
**제한:** 없음

## 💡 명령어 팁

### 자주 쓰는 조합

**1. 아침 루틴:**
```
/오늘레이드 → 오늘 일정 확인
```

**2. 주말 계획:**
```
/이번주레이드 → 이번 주 전체 확인
/추천생성 → AI 추천 받기
```

**3. 개인 관리:**
```
/내캐릭터 → 캐릭터 확인
/레이드설정 [캐릭터] [레이드들] → 설정
```

**4. API 키 관리:**
```
/api상태 → 키 상태 확인
/api목록 → 등록된 키 확인
```

### 단축키
명령어는 자동완성됩니다!
```
/오 → /오늘레이드
/이 → /이번주레이드
/내 → /내레이드, /내캐릭터, /내통계, /내알림
/api → /api추가, /api목록, /api상태, /api삭제
```

## ❓ 자주 묻는 질문

### Q. 명령어가 작동하지 않아요
A. 다음을 확인하세요:
- `/`로 시작했나요?
- 슬래시 명령어가 활성화되어 있나요?
- 봇이 채널을 볼 수 있나요?
- API 키가 등록되어 있나요?

### Q. 명령어를 잘못 입력했어요
A. 걱정 마세요! 다시 입력하시면 됩니다.

### Q. API 키 명령어는 누가 쓸 수 있나요?
A. 관리자 또는 길드 마스터만 사용 가능합니다.

### Q. 더 많은 명령어가 있나요?
A. `/도움말`을 입력하면 전체 목록을 볼 수 있습니다.
