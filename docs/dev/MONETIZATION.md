```

```

→ "예쁘게 보기: https://loil.kr/stats"

```

#### 광고 배치
```
홈페이지:
├── 상단 배너 (1개)
├── 사이드바 (1개)
└── 콘텐츠 중간 (선택)

페이지별:
- 레이드 정보 페이지 (높은 트래픽)
- 통계 대시보드
- 가이드 페이지
```

#### 예상 수익
```

CPM ₩1,500 기준
→ 월 ₩13,500

DAU 500명 (길드 50개)
→ 월 ₩67,500

DAU 2,000명 (길드 200개)
→ 월 ₩270,000
